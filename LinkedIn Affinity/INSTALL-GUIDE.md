# LinkedIn Affinity Extension - Installation Guide

## Quick Install (2 minutes)

### Step 1: Install the App

1. Open the downloaded **LinkedIn Affinity-X.X.dmg** file
2. Drag **LinkedIn Affinity** to the **Applications** folder
3. Eject the DMG (right-click > Eject)

### Step 2: Enable in Safari

1. Open **Safari**
2. Go to **Safari > Settings** (or press `Cmd + ,`)
3. Click the **Extensions** tab
4. Check the box next to **LinkedIn Affinity Extension**
5. Click **Turn On** when prompted

### Step 3: Configure (First Time Only)

1. Open a LinkedIn conversation page
2. Click the extension icon in Safari's toolbar
3. Enter your Affinity API key
4. Click "Test Connection" to verify

## Using the Extension

On any LinkedIn conversation:
- Click the **"Send to Affinity"** button, or
- Press **Cmd + Shift + A**

The conversation will be saved to Affinity CRM automatically.

## Getting Your Affinity API Key

1. Log in to [Affinity](https://www.affinity.co)
2. Go to **Settings > API**
3. Generate or copy your API key
4. Paste it in the extension settings

## Troubleshooting

### Extension not showing in Safari?
- Quit Safari completely and reopen
- Check Safari > Settings > Extensions again

### "Send to Affinity" button not appearing?
- Make sure the extension is enabled (Safari > Settings > Extensions)
- Refresh the LinkedIn page
- Check browser console for errors (Develop > Show JavaScript Console)

### API connection failed?
- Verify your API key is correct
- Check your internet connection
- Ensure you have access to the Affinity API
